# Sketch2Face AI

Sketch2Face AI is a Deep Learning-based project that converts hand-drawn facial sketches into realistic human face images using Generative AI and Computer Vision techniques.

This project demonstrates image-to-image translation where an AI model learns the relationship between sketch images and real face images to generate photorealistic outputs.

---

## 🚀 Features

- Convert sketches into realistic face images
- Deep Learning-based image generation
- Image preprocessing using OpenCV
- End-to-end implementation in Jupyter Notebook
- Visualization of generated outputs
- Simple and easy-to-understand workflow

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- TensorFlow / PyTorch
- OpenCV
- NumPy
- Matplotlib
- Deep Learning
- Computer Vision
- Generative AI
